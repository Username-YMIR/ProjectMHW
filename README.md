# ProjectMH
예시:
UE5 기반 3인칭 액션 RPG 게임, 몬스터 헌터 : 월드의 모작 프로젝트입니다

팀원
팀장 : 손승우
팀원 : 이건주, 허혁

## 개요

개요예시 : 몬스터 헌터 월드를 GAS 시스템을 접목해 구현한 게임입니다. 빠른 템포의 전투와 직관적인 조작감을 목표로 합니다.

## 기본 기능

예시:
- 캐릭터 이동
- 무기 교체
- 발사

## 구현 예정 기능

예시: 
- 달리기 (Sprint)
- 체력 시스템 + HUD

세부 구현 계획은 [Issues](../../issues)에서 확인할 수 있습니다.

## 브랜치 전략
예시: 
| 브랜치 | 용도 |
|--------|------|
| main | 안정된 버전 |
| feature/* | 기능 개발 |

## 커밋 컨벤션

```
feat: 새로운 기능 추가
fix: 버그 수정
asset: 에셋 추가/수정
refactor: 코드 정리
docs: 문서 수정
chore: 설정 파일 등 기타 수정
```
